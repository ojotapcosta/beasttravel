# Beast Travel Solver

A standalone, zero-dependency React application designed to solve the Beast Travel boat puzzle rows.

## Features
- **Premium Dark Mode UI**: Designed with Tailwind CSS.
- **Dynamic Matching**: Automatically calculates the required lengths of location strings and matches them against the 154 possible destinations from the extracted game lore.
- **Zero Config**: Due to macOS permission restrictions on this local environment regarding Node.js in the Documents folder, this was built as a zero-dependency React app via CDN.

## How to Run
Simply open `index.html` in any modern web browser. 

No `npm install` or node server required!

## Built With
- React 18 (CDN)
- Babel Standalone (JSX interpretation)
